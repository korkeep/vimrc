## Vimrc Config
**Step1**: apt 패키지 업데이트
```
sudo apt-get update
sudo apt-get upgrade
```
**Step2**: vimrc 설치 
```
sudo apt-get install vim
```
**Step3**: [jellybean 테마 설치](https://www.vim.org/scripts/script.php?script_id=2555) 
**Step4**: color 폴더에 테마 복사
```
cp <테마> /usr/share/vim/vimOO/color/
```
**Step5**: .vimrc 파일 수정
```
vi ~/.vimrc
vimrc.txt 파일의 내용 복사
```
